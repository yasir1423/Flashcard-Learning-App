Flash Card Learning App
Description: 
Allows the students to add flash cards, review them and takes a test and displays their result and save the progress.
Technologies Used:
1.Python
2.os module
3.json file handling
How to Run:
Install pyhton on your editor
Run:Main.py and ensure that flashcards.json is in the same folder.
